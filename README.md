## **Auctioneer** is a dynamic auction platform where users can list items for auction and place bids on items listed by others. Leveraging the skills developed over the past three semesters, this project aims to create a seamless auction website.

### Goal

The goal of this project is to utilize the front-end development skills acquired in previous courses to build an engaging and functional auction site.

## API

This project interfaces with the Noroff API under the Auction Endpoints. Full documentation of the API can be accessed here:

- [API Swagger Documentation] https://v2.api.noroff.dev/docs/static/index.html#/

### User Stories and Technical Requirements

### User Stories

- Users with `@stud.noroff.no` email can register.
- Registered users can log in, log out, and update their avatar.
- Registered users can view their total credit, create listings, and bid on items.
- Unregistered users can search through listings.

### Technical Restrictions

- **CSS Framework:** Must use Bootstrap (version >5.0.1), Tailwind (version >3.0.0), or MUI (version >5.11.8).
- **Hosting:** Must be hosted on GitHub Pages or Netlify.
- **Design Application:** Must use Adobe XD, Figma, or Sketch.
- **Planning Application:** Must use Trello or GitHub Projects.
